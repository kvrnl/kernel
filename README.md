# Kernel

**A fast, GPU-accelerated Windows terminal - by [KVRNL](https://kvrnl.io).**

Real shells (PowerShell, Command Prompt, Git Bash, WSL) with a built-in, always-available Bash, plus
one-click profiles for AI coding CLIs (Claude Code, Codex, Gemini) when they're installed. Tabs,
GPU rendering, themes, clickable links, and search - a terminal that actually looks and feels good.

### Features
- GPU-accelerated rendering (xterm.js WebGL) with true ConPTY shells
- Auto-detected shell profiles + a bundled Bash so bash commands always work
- One-click AI CLI profiles - Claude, Codex, Gemini - render full TUIs perfectly
- Tabs, clickable links, find, copy/paste, and a clean settings panel
- Themes, fonts, cursor, and window opacity - all configurable
